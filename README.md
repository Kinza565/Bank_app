# Banking Homework App

**Version:** 1.0.0  
**Tech Stack:** Python, FastAPI  

## Description
This is a simple banking application built for homework purposes using **FastAPI**.  
It uses **in-memory storage** to store user data and balances.  

Features:  
- User Authentication  
- Deposit Funds  
- Bank Transfer
